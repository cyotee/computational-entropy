- 01-foundations/
- 02-computational-models/
- 03-channels-and-entropy/
- 04-gravitational-channel/

- 06-synthesis/

Each paper subdir should contain:

- figures/

Do not duplicate explanatory text here — reference canonical files in foundations/, computational-models/, and emergent-gravity/.
