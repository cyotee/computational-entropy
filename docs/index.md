## Type safety (locked)

- **Load \(L\)** is a **dimensionless scalar** that clocks proper time.  
- **Structure metric \(G\)** is a **metric** (or edgewise cousin).  
- **\(L \neq G\)**. Discrete ledgers \(L^{\mathrm{disc}}\) are not continuum \(L\).
